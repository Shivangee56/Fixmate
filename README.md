# Fixmate

Fixmate is a MERN stack web application that helps users report and manage service issues efficiently.

## Features
- User authentication
- Issue reporting
- Backend API with Node.js and Express
- MongoDB database
- Responsive frontend

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Setup

Clone repo:

git clone https://github.com/Shivangee56/Fixmate.git

Install dependencies:

npm install

Run:

npm start
